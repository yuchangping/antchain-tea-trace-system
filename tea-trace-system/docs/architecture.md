# 系统架构设计

## 1. 文档目标

说明茶叶溯源系统的整体架构、分层职责和主要模块边界。

## 2. 系统分层

- 前端展示层
- 后端服务层
- 区块链层
- 数据库与文件存储层

## 3. 核心数据流

- 后台录入批次与溯源数据
- 后端写入 MySQL
- 后端生成摘要并发起上链
- 前端与扫码页读取展示结果

## 4. 模块职责

- `frontend/`：管理后台与扫码查询页面
- `backend/`：REST API、鉴权、上传、二维码、业务处理
- `blockchain/`：合约、部署脚本、链环境配置
- `docs/`：架构、接口、数据库、演示说明
